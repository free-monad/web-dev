Home

This page is showing you how the nav tag can be used to create a navigation bar. The nav tag is another example of a tag with semantic meaning. The tag itself does not provide layout information by default in most browsers, it simply informs readers of the page code that this section is for navigating around the webpage.

Notice that inside the nav tag is an unordered list. While this is not necessary in order to create a navbar, doing it this way is customary at this time. Before the nav tag existed, lists were a common way to create nav bars.
