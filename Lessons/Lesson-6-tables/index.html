This page is showing you how to work with tables. Tables are really useful for presenting rows of data to users. A long time ago they were also used to form the layout of a web page (because there was no styling available back then). Today they are used exclusively for presenting information, often repeating sets of data.

Below, you will see an example of a basic html table with data inside. It represents the income distribution of the United States by percentile. Make sure you look at the source code and notice the < table > tag. It contains several levels of tags inside of it. The first two children tags are < thead > and < tbody > . The grand children of these tags (the tags inside thead and tbody) are < th > and < tr > respectively. And then even further down (the great-grandchildren) are < td > tags. The tag tr represents a row in the table and each td tag represents a piece of information similar to a cell in a spreadsheet. The thead and tbody tags are optional but are useful for semantically organizing your table. Later on we will work on styling tables to make the information look much nicer. On modern pages, most table out lines are hidden so that you scarcely notice that the information you are viewing is contained in a table at all.
